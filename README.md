# Blockchain Basics
## Мобильно приложение для изучения основ блокчейн технологий

В приложении есть 8 курсов:
```
Что такое блокчейн
Виды криптовалют
Хэширование
Блоки транзакций
NFT Токен
Майнинг
Транзакции
Смарт контракты
```

А так же мини игры:
```
Угадай крипту
Блоки транзакций
Хэш-комбинация
Крипто ферма
Крипто казино
```
