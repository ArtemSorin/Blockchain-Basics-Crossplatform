# Blockchain Basics
## :fire:Мобильно приложение для изучения основ блокчейн технологий:boom:

> Смена аватарки, cбор достижений, просмотр страницы топ 5 пользователей

:dizzy: В приложении есть 8 курсов :star2::
```
Что такое блокчейн
Виды криптовалют

Блоки транзакций
NFT Токен
Майнинг
Транзакции
Смарт контракты
```
